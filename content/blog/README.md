# Blog Content

Add `.mdx` files to this folder to publish articles.

## Frontmatter example

```yaml
---
title: "Your Article Title"
description: "A short summary for SEO and cards."
date: "2026-07-03"
updated: "2026-07-03"
author: "IG Downloader Team"
category: "Guides"
tags: ["instagram", "reels", "download"]
featured: false
popular: false
image: "/images/blog/your-cover.jpg"
faq:
  - question: "Optional FAQ question?"
    answer: "Optional FAQ answer for schema markup."
draft: false
---
```

Your MDX body starts below the frontmatter.
